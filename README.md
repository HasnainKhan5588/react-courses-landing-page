# React Courses Landing Page

A simple and modern **Courses Landing Page** built using **React, Vite, and Tailwind CSS**.
This project displays different courses in a clean layout with a responsive slider.

## 🚀 Live Demo

* **Vercel:** https://react-courses-landing-page.vercel.app/
* **GitHub Repo:** https://github.com/HasnainKhan5588/react-courses-landing-page

## ✨ Features

* Responsive design for all screen sizes
* Courses slider using **react-slick**
* Clean and modern UI
* Reusable course card components
* Simple and beginner-friendly React structure

## 🛠️ Technologies Used

* React
* Vite
* Tailwind CSS
* React Slick
* JavaScript (ES6)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/HasnainKhan5588/react-courses-landing-page.git
```

Go to the project folder:

```bash
cd react-courses-landing-page
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
